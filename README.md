# RAG-mobile_app

## Описание проекта

AI Knowledge Hub — информационный веб-сайт, разработанный в рамках проектной деятельности по теме «Автоматизированное обучение RAG-моделей». Сайт предназначен для представления целей, задач, архитектуры и результатов проекта по созданию корпоративной платформы AI-as-a-Service для интеллектуального поиска знаний и обработки документов.

Проект демонстрирует возможности применения технологий Retrieval-Augmented Generation (RAG) для организации эффективного поиска информации и взаимодействия пользователей с корпоративной базой знаний.

---

## Цель проекта

Создание информационного веб-ресурса, отражающего основные аспекты разработки сервиса автоматизированного обучения RAG-моделей и демонстрирующего архитектуру предлагаемого решения.

---

## Задачи проекта

* Исследование предметной области искусственного интеллекта и технологий RAG.
* Анализ существующих решений для интеллектуального поиска информации.
* Разработка структуры веб-сайта и пользовательского интерфейса.
* Представление архитектуры программного решения.
* Формирование журнала этапов реализации проекта.
* Подготовка информационных материалов и проектной документации.

---

## Структура сайта

Сайт состоит из следующих разделов:

### Журнал проекта

### Полезные ресурсы

Ссылки на технологии и программные продукты, использованные в проекте.

### Паспорт проекта

Основные сведения о проекте: цель, задачи, предметная область и ожидаемые результаты.

## Архитектура решения

Предлагаемая архитектура включает следующие компоненты:

1. Загрузка и обработка документов.
2. Формирование эмбеддингов.
3. Сохранение данных в векторной базе ChromaDB.
4. Поиск релевантных фрагментов информации.
5. Генерация ответов с использованием языковой модели Gemma через Ollama.
6. Взаимодействие пользователя через OpenWebUI.

## Запуск проекта

Для запуска сайта необходимо:

1. Скачать проект на локальный компьютер.
2. Открыть файл `index.html` в браузере.

Дополнительная установка программного обеспечения не требуется.
